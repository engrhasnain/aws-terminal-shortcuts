Enable Ctrl+Shift+C and Ctrl+Shift+V shortcuts in AWS EC2 Instance Connect

‚úì What This Extension Does:
Adds keyboard shortcuts (Ctrl+Shift+C and Ctrl+Shift+V) to AWS EC2 Instance Connect terminal, eliminating the need to right-click for copy/paste operations. Regular Ctrl+C still works for stopping processes!
1

Prepare Your Files

Make sure you have all the extension files in one folder:

  • manifest.json
  • content.js
  • icon16.png, icon48.png, icon128.png
  • README.md (optional)
2

Open Extensions Page

Navigate to your browser's extensions page:

Chrome: chrome://extensions/
Opera: opera://extensions/

Just copy the URL above and paste it in your browser's address bar.

3

Enable Developer Mode

Look for a toggle switch labeled "Developer mode" in the top-right corner of the extensions page and turn it ON.

üí° Tip: This is required to load unpacked extensions. It's completely safe and only affects your ability to load local extensions.
4

Load the Extension

Click the "Load unpacked" button that appears after enabling Developer mode, then:

  • Navigate to the folder containing your extension files
  • Select the folder (not individual files)
  • Click "Select Folder" or "Open"

The extension will now appear in your extensions list!

5

Grant Permissions

The extension needs clipboard permissions to work. When prompted:

  • Allow clipboard read permission (for paste)
  • Allow clipboard write permission (for copy)
‚ö†Ô∏è Privacy Note: These permissions only work on AWS Console pages. Your clipboard data stays completely private and local to your browser.
6

Test It Out!

Navigate to your AWS EC2 Instance Connect terminal:

  1. Go to AWS Console ‚Üí EC2 ‚Üí Instances
  2. Select an instance and click "Connect"
  3. Choose "EC2 Instance Connect"
  4. Wait for the terminal to load
  5. Look for "AWS Terminal shortcuts active!" notification

üìã Keyboard Shortcuts

Ctrl + Shift + C
Copy selected text from terminal
Ctrl + Shift + V
Paste text into terminal
üí° Important: Regular Ctrl+C still works as the interrupt signal to stop processes in the terminal. This extension uses Ctrl+Shift combinations to avoid conflicts!

üîß Troubleshooting

  • Extension not working? Make sure it's enabled in chrome://extensions/
  • No notifications? Refresh the AWS terminal page
  • Paste not working? Check clipboard permissions in extension details
  • Still issues? Press F12 and check Console tab for error messages
